# Documentación principal 
Esta documentación pertenece al proyecto de código libre **Automatización de Archivos** que pertenece un portafolio de trabajo.

## ¿Que es lo que hace?
Dentro del [README](../README.md) explicamos los básico del proyecto. Pero vamos a aquí explicamos alguna de las características de este.

- **Contiene funciones de comprobación de errores** para evitar errores de archivos repetidos y errores de sintaxis dentro de nuestro archivo de configuración.
- **Puedes agregar cualquier organización de cualquier archivo por su extensión** puede ser una extensión como `.pdf` o `pdf`. Si es necesario también puedes decir que carpetas puede depender cada extensión de archivos (algo que implementaremos después de un tiempo próximamente).
